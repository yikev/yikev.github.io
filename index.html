About Me

I’m a software engineer who builds full-stack applications and performance-minded systems. Here’s a quick summary of my work.

  • Digital Asset Management (DAM) System - Enterprise-ready DAM with ASP.NET Core, EF Core, Azure Blob Storage, and Next.js/React. Secure uploads, role-based access, audit logging, advanced search, and admin dashboards.
  • AI-Powered E-Commerce Analytics (Shoplytics) - Next.js + FastAPI app that ingests Shopify data. XGBoost forecasting predicts demand, detects anomalies, and powers insight dashboards with Mantine UI.
  • Eternal Gauntlet - C++ 2D turn-based game using a custom ECS, adaptive AI combat, and procedurally generated maps.
  • Multi-Tenant File Sharing Platform - FastAPI + Postgres + React with JWT auth, tenant-scoped Cloudinary storage, role-based permissions, and audit trails.
  • Blackjack Trainer - React + TypeScript trainer for multi-deck blackjack with strategy feedback, split support, and session stats.
