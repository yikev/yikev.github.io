Full stack analytics for Shopify data - demand forecasting, anomaly detection, and actionable insights.

Overview

Shoplytics connects to the Shopify API and turns raw store activity into forecasts and insights. I designed an XGBoost based model that learns seasonality and recent trends to predict daily sales and flag outliers. The dashboard uses Mantine UI and React for interactive charts and product level breakdowns that help merchants plan inventory and marketing.

Highlights

  • XGBoost forecasting trained on seasonality, recent performance, and promotions.
  • Anomaly detection to alert on unexpected spikes or dips in sales.
  • Product and collection comparisons with trend lines and rolling windows.
  • Account scoped auth with JWT and rate limited backend APIs.
  • Downloadable reports with recent vs predicted performance.

Tech Stack

  • Frontend - Next.js, React, Mantine UI.
  • Backend - FastAPI, Python, pandas, scikit-learn, XGBoost.
  • Data - Shopify REST/GraphQL API, Postgres or SQLite for dev.
  • Infra - JWT auth, task queue for ETL and model refresh.

Architecture

The ETL service ingests orders, products, and customers from Shopify and stores curated tables. Feature builders compute rolling 3 to 5 day windows, season averages, volatility, and promotion flags. The model service exposes forecast and anomaly endpoints that the Next.js app calls to render charts and tiles.

Key Endpoints

  • POST /api/ingest/run - pull latest Shopify data.
  • GET /api/forecast/daily?store_id=... - return next N day predictions.
  • GET /api/anomalies?range=... - list flagged anomalies with reasons.
  • GET /api/products/compare?ids=... - comparative metrics for products.

UI Features

  • Actual vs predicted charts with tooltips and zoom.
  • Recent performance tiles - revenue, AOV, conversion rate.
  • Filters for date range, channel, and collection.
  • Export to CSV or PNG for reports.

What I Built

  • Implemented ETL pipelines and schema for orders, products, and customers.
  • Engineered features - rolling windows, seasonality, and volatility metrics.
  • Trained and evaluated the XGBoost model with backtesting.
  • Developed FastAPI endpoints and auth guards.
  • Built the React dashboard with Mantine components and charts.
