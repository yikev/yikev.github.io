Blackjack Trainer

Interactive React + TypeScript app that teaches card counting and optimal decisions.

  • GitHub
  • Back to Projects

Overview

A training tool for learning blackjack fundamentals and card counting. It simulates multi deck shoes, betting, splits, and dealer rules, then scores your play against a strategy table. The UI surfaces immediate feedback and keeps session stats so users can track progress.

Highlights

  • Multiple decks with configurable penetration and shuffle rules.
  • Full support for hit, stand, double, split, and surrender options.
  • Basic strategy engine with correctness feedback on each decision.
  • Running and true count calculations with training prompts.
  • Session stats: hands played, win rate, EV, errors by category.

Tech Stack

  • Frontend - React + TypeScript.
  • State - React hooks, context for table state.
  • Build - Vite, GitHub Pages deployment.
  • Testing - Vitest for core logic.

Architecture

Core logic is a pure TypeScript engine that models the shoe, hand objects, dealer rules, and strategy evaluation. The UI subscribes to engine events to render the table, chips, and decision prompts. A stats module logs outcomes and produces summaries used for progress charts.

Key Modules

  • shoe.ts - deck generation, shuffles, penetration, burn.
  • hand.ts - hand values, soft vs hard totals, splits.
  • strategy.ts - correctness checks and hints.
  • trainer.ts - session loop and scoring.

UI Features

  • Keyboard shortcuts for fast training.
  • Overlay feedback when decisions are optimal or sub optimal.
  • Charts for recent accuracy and EV by hand type.

What I Built

  • Added split support and betting flows with chip stacks.
  • Built a strategy checker and feedback overlays.
  • Deployed via GitHub Pages with Vite config fixes.
